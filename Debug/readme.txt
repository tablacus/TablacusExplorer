A tabbed file manager with Add-on support.
Tablacus Explorer

Author: Gaku


Features in Tablacus Explorer:

* Tabbed interface
* Add-on support
* Completely Portable, No installation required
* Multiple language and Unicode support
* Customizable association, menus, keys, mouse gestures, alias
* Open source
* 64-bit(TE64.exe/x64 Edition) and 32-bit(TE32.exe/x86 Edition) versions are available


Set up:

* Extract the ZIP file to any folder
* Run Tablacus Explorer (te64.exe/te32.exe)

- In case you installed Tablacus Explorer to the Program Files directory:

* Settings are saved in "%AppData%\tablacus\explorer\config".
* Open Options
* Press "Initialize config folder"


Uninstall:

* Delete the tablacus folder.

- In case you installed Tablacus Explorer to the Program Files directory:

* Delete the folder "%AppData%\tablacus" too.
